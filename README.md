# Ed Witten Bot
This bot runs on the official Discord server of the Physics Student Union at the University of Toronto. It performs verification of new members, handles various routine tasks, and has more easter eggs than a FNaF game. Feel free to adapt it to your own server or contribute to its development! See DOCUMENTATION.md for many details.

Note to future maintainers: whatever you do, do not commit .env.
